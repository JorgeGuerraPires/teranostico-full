Keep in mind that what you edit here and save will replace the original content in the database, therefore, use this tool with caution. Do not forget to inform any user whose information is altered.

Instructions

Reset password

Once you reset the password, it should appear a token or/and link on the user information (you may need to refresh the user table and reload the user on the edit table): send this token/link to the user

Important. Just accept requests to reset password from e-mails associated with the user account, and just send the token to the associated e-mail with this account. If necessary, gather more information before reseting the password. Once the password is reset, the user will no longer be able to use the previous password.

Deactivate/activate accounts

On the current design, all the registered users are initially set as deactivated.
Once you activate an account, the user will have access to it!
If you deactive an user, they will lose access to the account!
